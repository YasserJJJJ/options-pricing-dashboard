# Options Pricing Dashboard

An interactive quantitative finance dashboard built with Python and Streamlit.  
The app uses the Black-Scholes model to price European call and put options, calculate Greeks, and visualize option sensitivity.

## Features

- Black-Scholes option pricing
- European call and put support
- Greeks calculation:
  - Delta
  - Gamma
  - Theta
  - Vega
  - Rho
- Payoff and profit/loss chart
- Option price sensitivity to stock price
- Greeks sensitivity chart
- Volatility sensitivity chart
- Clean Streamlit dashboard interface

## Tech Stack

- Python
- Streamlit
- Pandas
- Black-Scholes model
